# Blockchair downloads

```bash
# Latest Bitcoin addresses dump (compressed ~15 GB, decompressed ~150 GB)
./blockchair-dl -c 16 -d \
  https://gz.blockchair.com/bitcoin/addresses/blockchair_bitcoin_addresses_latest.tsv.gz
```

## Other useful URLs

| Dataset | URL |
|---------|-----|
| Addresses (latest) | `https://gz.blockchair.com/bitcoin/addresses/blockchair_bitcoin_addresses_latest.tsv.gz` |
| Outputs by date | `https://gz.blockchair.com/bitcoin/outputs/blockchair_bitcoin_outputs_YYYYMMDD.tsv.gz` |
| Transactions by date | `https://gz.blockchair.com/bitcoin/transactions/blockchair_bitcoin_transactions_YYYYMMDD.tsv.gz` |
| Blocks | `https://gz.blockchair.com/bitcoin/blocks/blockchair_bitcoin_blocks_latest.tsv.gz` |
